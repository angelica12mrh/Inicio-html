HTML

HTML, siglas de Hyper Text Markup Language, hace referencia al lenguaje de marcado para la
elaboración de páginas web. Es un estándar que sirve de referencia para la elaboración de páginas
Web en sus diferentes versiones, define una estructura básica y un código.
Fuente: HTML Wikipedia


HTTP

Hypertext Transfer Protocol o HTTP “En español protocolo de transferencia de hipertexto”, es el
protocolo usado en cada transacción de la World Wide Web, HTTP fue desarrollado por el World
Wide Web Consortium y la Internet Enginneering Task Force.
Fuente: HTTP Wikipedia

URL

Un localizador de recursos uniforme o URL Uniform Resource Location es un identificador de
recursos uniformes cuyos recursos referidos pueden cambiar, esto es, la dirección puede apuntar
a recursos variables en el tiempo.
